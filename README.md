  <h1>Política de Privacidade</h1>
  <p class="muted"><strong>Chatbot Arquivo Geral FURG</strong> — Última atualização: <span id="date"></span></p>

  <h2>1) Objetivo</h2>
  <p>Este documento descreve como o Chatbot Arquivo Geral FURG trata dados pessoais em interações realizadas via WhatsApp.</p>

  <h2>2) Quais dados podem ser tratados</h2>
  <ul>
    <li>Número de telefone (identificador do WhatsApp).</li>
    <li>Nome de perfil do WhatsApp (quando enviado pela plataforma).</li>
    <li>Conteúdo das mensagens enviadas pelo usuário.</li>
    <li>Metadados técnicos (data/hora, IDs de mensagem e status de entrega).</li>
  </ul>

  <h2>3) Finalidade do tratamento</h2>
  <ul>
    <li>Responder dúvidas e fornecer informações sobre horários, documentos e atendimento.</li>
    <li>Direcionar solicitações para atendimento humano quando necessário.</li>
    <li>Manter registros mínimos para auditoria técnica e melhoria do serviço.</li>
  </ul>

  <h2>4) Base legal (LGPD)</h2>
  <p>O tratamento pode ocorrer para execução de políticas/serviços solicitados pelo usuário e/ou legítimo interesse, respeitando a LGPD e boas práticas de segurança.</p>

  <h2>5) Compartilhamento</h2>
  <p>As mensagens trafegam pela infraestrutura do WhatsApp/Meta. Não compartilhamos dados com terceiros além do necessário para o funcionamento do serviço.</p>

  <h2>6) Retenção</h2>
  <p>Os dados podem ser armazenados pelo tempo necessário para a operação e segurança do serviço. Registros podem ser removidos mediante solicitação, quando aplicável.</p>

  <h2>7) Segurança</h2>
  <p>Adotamos medidas técnicas e organizacionais para proteger os dados contra acesso não autorizado, perda e uso indevido.</p>

  <h2>8) Direitos do titular</h2>
  <p>Você pode solicitar informações, correção e exclusão de dados, quando aplicável, conforme a LGPD.</p>

  <h2>9) Contato</h2>
  <p>
    Para solicitações relacionadas à privacidade, entre em contato pelo e-mail:
    <strong>arquivo.geral@furg.br</strong>.
  </p>
